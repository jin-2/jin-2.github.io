### 보편적인 문법
    @font-face {
        font-family: fontname;
        src: url('filename.eot');
        src: local('☺︎'),
        url('filename.woff') format('woff'),
        url('filename.ttf') format('truetype'),
        url('filename.svg#filename') format('svg');
    }

- WOFF: Web Open Font Format
- TTF: True Type Font
- EOT: Embedded Open Type
